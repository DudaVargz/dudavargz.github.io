## ✨ Sobre o projeto

Este portfólio foi desenvolvido do zero com foco em design criativo e responsividade. É a minha vitrine profissional para recrutadores e pessoas interessadas no meu trabalho.

**Seções incluídas:**
- **Hero** — apresentação com animações
- **Sobre mim** — resumo pessoal e soft skills
- **Formação** — linha do tempo acadêmica
- **Habilidades** — tecnologias e ferramentas
- **Projetos** — trabalhos desenvolvidos
- **Contato** — formas de entrar em contato
